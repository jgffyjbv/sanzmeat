# Sanz Klausenburg Fleish — Landing Page

Static landing page for **Sanz Klausenburg Fleish**, a kosher butcher shop at
4720 New Utrecht Ave, Brooklyn, NY 11219 · (718) 437-4900 · sanzmeat47@gmail.com

## Files
- `index.html` — landing page (hero, about, gallery, contact form)
- `privacy.html` — privacy policy
- `styles.css` — styles
- `images/` — logo + photos

## Contact form
Submits via [FormSubmit](https://formsubmit.co) to **sanzmeat47@gmail.com**.
The first submission triggers a one-time confirmation email from FormSubmit —
click the link in it once to activate delivery.

## Photos
`images/storefront.jpg`, `images/meat-case.jpg`, and `images/gallery-3.jpg` are
placeholders. Replace them with real photos (same filenames) to go live with real imagery.

## Deploy
Hosted on GitHub Pages from the `main` branch.
